About This Project

AI-powered Food Image Classification System

What is this project?

This is an AI-based Food Image Classification System that uses Convolutional Neural Networks (CNN) and Transfer Learning to classify food images into 34 different categories. It provides instant nutritional insights for each predicted food item, making it a powerful tool for dietary awareness and food recognition.

What does it do?

The system accepts a food image as input and classifies it into one of 34 food categories. Along with the prediction, it provides:

  • Predicted food class with confidence score
  • Top 5 prediction probabilities
  • Detailed nutritional information (calories, protein, carbs, fat, fiber, vitamins, minerals)
  • Model performance metrics (accuracy, precision, recall, F1-score)
  • Redis-cached results for faster repeated lookups

How was it developed?

Model Training:

  • Trained using TensorFlow / Keras deep learning framework
  • 3 models: Custom CNN, VGG16 (Transfer Learning), ResNet (Transfer Learning)
  • Data augmentation for robust training
  • Comprehensive validation with TP, TN, FP, FN metrics

Application Architecture:

  • Implemented using Object-Oriented Programming principles
  • 6 modular utility classes (ModelLoader, ImagePreprocessor, Predictor, MetricsReader, NutritionLoader, RedisCache)
  • Integrated Redis caching for performance optimization
  • Built with Flask web framework
  • Deployed using Vercel for production hosting

Technology Stack

Python 3.x TensorFlow / Keras Flask Redis HTML5 CSS3 JavaScript Bootstrap 5 NumPy Pillow Vercel Git / GitHub

Model Architectures

  • Custom CNN: A custom-built convolutional neural network designed and trained from scratch for food classification.
  • VGG16: Pre-trained VGG16 model with transfer learning, fine-tuned on our food dataset for improved accuracy.
  • ResNet: Pre-trained ResNet model with transfer learning, leveraging residual connections for deep feature extraction.
